# KeySense 快捷键提示器 - 手动测试流程

> 适用于测试工程师、QA 和最终用户验证。建议在真实环境中执行。

---

## 🔧 测试前准备

1. **安装依赖**：确保项目已安装
   ```bash
   cd /mnt/d/Montarius/source/KeySense
   npm install
   ```

2. **启动开发版**：
   ```bash
   npm start
   ```

3. **确认应用状态**：
   - 任务栏托盘图标可见
   - 按 `Ctrl+Shift+G` 可唤出窗口
   - 窗口半透明（0.1）隐藏，点击托盘或快捷键可显示（0.9）

4. **准备测试环境**：
   - 开启多个显示器（如有）
   - 启动测试应用：WinDbg、Chrome、Word、WPS、微信
   - 确保系统无其他快捷键冲突
